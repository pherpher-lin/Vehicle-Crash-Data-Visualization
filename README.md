# [Vehicle-Crash-Data-Visualization](https://public.tableau.com/app/profile/christopher.lin1994/viz/ChristopherLinVizProject/RiskandLocation)

### About the Data
[NYC Crash Mapper](https://crashmapper.org/#/)
* NYC Crash Mapper is an interactive web tool that maps and visualizes vehicle collision data from NYPD (New York City Police Department), covering crashes from August 2011 to the present. 
* The tool was developed so that users (researchers, activists, the public) can explore crash data by location, type of crash (injury, fatality, pedestrian, cyclist, motorist), time range, and other filters.
